# Configuração

No seu arquivo de configuração `app/config/config.yml` adicione a seguinte entrada:

```yml
# app/config/config.yml

# ...
cekurte_mailer:
    mail2easy:
        username: usuario-do-mail2easy
        password: senha-do-mail2easy
```
